# Сопутствующие дополнения  
для работы данного чата и всех сопутствующих файлов вам будет необходим ZeroTier либо же белый/стабильный ip.
ZeroTier вы можете скачать по ссылке: https://www.zerotier.com
приложение идиально работает в локальной сети а если ваше желание устремляется к работе через мобильные сети или же инные не как не связанные точки сети wifi то вам нужен ZeroTier который будет установлен на всех устройствах которые должны быть связаны меж собой при желании вы можете заменить ZeroTier на hamachi или же вовсе обойтись без сторонних приложений(только при наличи неизменного ip для серверной части)
## Требования
для сборки приложения вам необходимо иметь go 1.20+ и gitBash.

# Инструкция по сборке и запуску
для сбоки приложения необходимо через комманду cd перейти в отдельные именные части кода(каждая такая часть желательно в новом окне bash) а именно: "client", "relay_server" и "signaling_server".

для полноценной работы вам нужно заменить закоментированный текст с указанием куда надо вставлять ваш ip в файле main.go в папке client.

команды для сборки файлов в .exe формат производятся через bash находясь в конкретной папке:
```bash
go build -o client.exe 
go build -o relayserver.exe 
go build -o signaling_server.exe
//далее советую первый раз советую запускать relay и signaling через окна bash что бы вы могли узнать почему конкретно появились ошибки если они конечно есть.
./ relay_server.exe
./ signaling_server.exe 
// client сразу выводит командною строку так что нет необходимости работать с ним только через bash вообще. 

```

# trouble shooting:
если вы столкнулись с преблемой в виде потери или каких бы то нибыло проблем с зависимостями введите команыды: 
```bash
go mod tidi 
go mod init
```
так же может встретиться некорекная работа приложения при включённом брандмауэр так что если сообщения не доходят попробуйте его отключить так как он пресикает большую часть попыток связаться с устройством из вне.
