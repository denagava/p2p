это секретный чат с максимальной для лично моих возможностей реальтзацией защиты и шифровки.
для работы данного чата и всех сопутствующих файлов вам будет необходим ZeroTier либо же белый/стабильный ip.
ZeroTier вы можете скачать по ссылке: https://www.zerotier.com
приложение идиально работает в локальной сети а если ваше желание устремляется к работе через мобильные сети или же инные не как не связанные точки сети wifi то вам нужен ZeroTier который будет установлен на всех устройствах которые должны быть связаны меж собой при желании вы можете заменить ZeroTier на hamachi или же вовсе обойтись без сторонних приложений(только при наличи неизменного ip)
для сборки приложения вам необходимо иметь go 1.21 и выше и gitBash.
теперь точная инструкция по сборке и запуску:
для сбоки приложения необходимо через комманду cd перейти в отдельные именные части кода(каждая такая часть желательно в новом окне bash) а именно: "client", "relay_server" и "signaling_server".
для полноценной работы вам нужно заменить закоментированный текст с указанием куда надо вставлять ваш ip в файле main.go в папке client.
команды для сборки файлов в .exe формат производятся через bash находясь в конкретной папке:
go build -o "имя файла".exe 
trouble shooting:
если вы столкнулись с преблемой в виде потери или каких бы то нибыло проблем с зависимостями введите команыды: go mod tidi и go mod init
